Code for Vilnius

Kas tai yra?

Code for Vilnius – tai iniciatyva, kurios metu susiburia programuotojai, analitikai, dizaineriai, vartotojo sąsajos ir kitų sričių (nuo urbanistikos iki žurnalistų) specialistai, kurie nori padėti išspręsti Vilniaus problemas, kartu, skaidriai ir efektyviai.

Ši iniciatyva buvo pasiūlyta aktyvaus vilniečio Renat Buga, kuris sudalyvavęs analogiškose iniciatyvose JAV, sumanė panašiu būdu padėti Vilniui.

Code for Vilnius nėra vienkartinis renginys ar hackatonas, čia sprendžiamos svarbios miestui problemos. Pasinaudojus technologijomis bandome pagerinti vilniečių gyvenimą.

Prie renginių pastoviai prisideda savivaldybės ir jos įmonių atstovai, politikai, taip pat ir verslo bei nevyriausybinių organizacijų nariai. Visas darbas atliekamas savanoriškais pagrindais ir visiems sprendimams naudojamos atviro kodo technologijos.
